--------------------------------------------------------------------------------

  s0:  Input register
  s1:  x * INV_L_64
  s2:  Float-to-int conversion (N, J, M)
  s3:  Int-to-float conversion (N_fp)
  s4:  N_fp * LOG2_BY_64
  s5:  r = x - t (reduced argument)
  s6:  r2 = r * r
  s7:  r3 = r2 * r
  s8:  r4 = r2 * r2
  s9:  t2 = A2_64 * r2
  s10: t3 = A3_64 * r3
  s11: t4 = A4_64 * r4
  s12: sum23 = t2 + t3
  s13: sum234 = sum23 + t4
  s14: p = r + sum234
  s15: one_plus_p = 1.0 + p
  s16: core = T[J] * one_plus_p
  s17: Exponent adjustment (2^M scaling)
  s18: Output register
